Математика строится как формальная система:

    Аксиомы — принимаются без доказательства
    Правила вывода — как из аксиом получать теоремы
    Теоремы — всё, что выводимо из аксиом
    
    
7 правил Цермело (очень просто):
1. Правило равенства

    Два множества равны, если у них одинаковые элементы.

Пример: {яблоко, груша} = {груша, яблоко} (порядок не важен)
2. Правило пары

    Из любых двух объектов можно сделать множество-пару.

Пример: Из яблоко и груша → {яблоко, груша}
3. Правило объединения

    Из "мешка с мешками" можно высыпать всё в один большой мешок.

Пример: Из {{a,b}, {c}} → {a, b, c}
4. Правило всех подмножеств

    Из множества можно сделать "коллекцию всех возможных выборок" из него.

Пример: Из {a, b} → {∅, {a}, {b}, {a,b}}
5. Правило бесконечности

    Существует хотя бы одно бесконечное множество.

Конкретно: {∅, {∅}, {{∅}}, ...} — можно добавлять элементы бесконечно.
6. Правило вырезания (самое важное!)

    Можно взять существующее множество и оставить в нём только элементы с нужным свойством.

Пример: Из все_люди вырезать {x | x — блондин} → все_блондины

Но нельзя: {x | x — множество, которое не содержит себя} ← это парадокс Рассела!
7. Правило замены

    Если у каждого элемента поставить ему в пару другой объект, то получится новое множество.

Пример: Каждому числу n поставим в пару n+1:
Из {1,2,3} → {2,3,4}
