Семантика — ∀ = «для всех», ∃ = «существует».

Законы преобразования кванторных формул

    Отрицание кванторов (де-Морган для кванторов):
    ¬∀xA ≡ ∃x¬A
    ¬∃xA ≡ ∀x¬A

    Дистрибутивность ∀ над ∧:
    ∀x(A∧B) ≡ ∀xA ∧ ∀xB

    Дистрибутивность ∃ над ∨:
    ∃x(A∨B) ≡ ∃xA ∨ ∃xB


    Перестановка одноимённых кванторов:
    ∀x∀yA ≡ ∀y∀xA

    ∃x∃yA ≡ ∃y∃xA
    
    




ДНФ (дизъюнктивная нормальная форма)

Формула имеет вид дизъюнкции конъюнкций литералов (переменных или их отрицаний).

Пример:
(p∧¬q)∨(r∧s)∨(¬p∧q)

Построение: по таблице истинности берём строки, где формула = 1, для каждой строки составляем конъюнкцию литералов, затем соединяем их дизъюнкцией.
КНФ (конъюнктивная нормальная форма)

Формула имеет вид конъюнкции дизъюнкций литералов.

Пример:
(p∨¬q)∧(r∨s)∧(¬p∨q)

Построение: по таблице истинности берём строки, где формула = 0, для каждой строки составляем дизъюнкцию литералов (инвертированных), затем соединяем их конъюнкцией.
    

Правило экзистенциальной конкретизации (или введения/инстанцирования экзистенциального квантора, Existential Instantiation) — это логический вывод, который гласит: если мы знаем, что существует объект с определенным свойством (\(\exists xP(x)\)), то мы можем ввести имя нового конкретного объекта (обозначим его, например, c), для которого это свойство выполняется (\(P(c)\)), при условии, что c не использовался ранее и не будет использоваться в других посылках, чтобы избежать ложных заключений
